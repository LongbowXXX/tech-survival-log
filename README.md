# tech-survival-log 🛡️

**AI 時代を生き抜くための、ベテランエンジニアの技術生存戦略ログ。**

このリポジトリは、技術情報共有サービス [Zenn](https://zenn.dev/) の記事コンテンツを管理しています。

## 📝 Concept

**「AI に仕事を奪われるのではなく、AI を使い倒して進化する」**

業界歴 20 年以上のシステムエンジニアが、これまでの経験（C++ / 組み込み）を土台にしつつ、モダンな技術（LLM / Python / Kotlin / TypeScript）を活用していく様を記録します。

このブログ記事自体も、AI を活用した効率的なワークフローで執筆していきます。
